## Cursol control using imu data

### BLE server

To setup:
[ble5-nrf52-mac](https://github.com/tomasero/ble5-nrf52-mac)

Run:

    python button.py

For multiple:

    python buttons.py

### Cursor control 

- Install CocoaPods and run 'pod install'
- Open 'cursorCtrl.xcworkspace'
- Send data via UDP
- Run!

### Target genrator
    
    cd nav
    open index.html
